# 드림 리멤버
# 프로젝트 특징

- 스프링부트를 기반으로 CRUD 기능이 포함된 웹서비스
- AWS에 스프링부트 프로젝트 배포

## 개요

- 명칭 : 리멤버 드림
- 개발 인원 : 1명
- 개발 기간 : 2021.03.20 ~ 2021. 03.22
- 주요 기능 : 전체 게시글 조회, 게시글 CRUD
- 개발 언어 : html, css, javascript, java 8
- 개발 환경 : springboot 2.4.4, jpa
- 데이터베이스 : mysql
- 형상관리 툴 : git
- 간단 소개 : 꿈(진짜 밤에 꾼 꿈)을 기록하는 커뮤니티
